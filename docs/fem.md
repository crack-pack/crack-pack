# Fallen Empires (`fem`) — known gaps & assumptions

Part of [crack-pack](../README.md) · see the [status matrix](../ASSUMPTIONS.md#status-matrix).

**Status:** sheet data ✅ validated · collation 🔴 simplified.

## What's solid
- Two 11×11 sheets (common + uncommon; no rare sheet), grids checksum-validated:
  common merges to 36 base cards (15×C4 + 20×C3 + Delif's Cone ×1); uncommon
  25×U3 + 5×U2 + 36×U1.
- **Multi-art commons**: a common has a distinct artwork per appearance
  (C4 → four, C3 → three), notated `(A)`–`(D)`.

## The gap
- Like The Dark, the **uncommon sheet may or may not be split depending on the
  box**, so there is no single deterministic collation to reproduce.

## What we assumed
- A plain two-sheet striped model with stripe cycle **`[2,3,4,5]`** — a
  deliberate simplification. Grids are real; the pack grouping is a stand-in.

## How you can help
- Box openings that pin down the uncommon split behaviour, and the real stripe
  widths.
